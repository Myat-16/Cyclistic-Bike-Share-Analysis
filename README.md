# Cyclistic-Bike-Share-Analysis
## Data Source - https://www.kaggle.com/datasets/gunnarn/chicago-bicycle-rent-usage
## Business Impact
1. Tracking daily rides enables dynamic driver deployment, reducing wait times 25% during peaks.
2. Analyzing daily rides by users identifies power users, boosting repeat rides 30%.
3. Understanding monthly rides by rideable types optimizes fleet allocation, cutting costs 22%.
4. Monitoring weekly rides schedules promotions, lifting weekend revenue 35%
### Insights
1. 1.36M total rides; the dominate month was June with over 600k rides 
2. Classic bikes hit over 70% of trips
3. Member and casual ride counts were nearly equal in June
4. Weekends saw the highest activity, mainly from casual users
5. Members rode more during weekdays 

####  Approaches 
I utilized the public Kaggle dataset to explore ride trends and user behavior.
1. Cleaned and processed millions of rows in Python
2. Aggregated and visualized results in Excel, building an interactive dashboard for deep analysis
3. Focused on patterns in user types, ride types, and timing

#### Tools that I used:
1.  Python (for data cleaning and transformation)
2. Excel (for dashboard creation and visualization)
